# Frontend

This directory contains the Svelte frontend application for oinkerui.

## Structure

- `src/` - Svelte application source code
  - Components, routes, stores, and utilities
- `public/` - Static assets (images, fonts, etc.)
- `tests/` - Frontend unit and integration tests

## Technology Stack

- **Framework**: Svelte with Vite
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
- **Testing**: Vitest (to be configured)

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Run tests
npm test
```

## Phase Implementation

This frontend will be implemented across multiple phases:
- **Phase 1**: Basic project structure and chat interface
- **Phase 2**: Enhanced UI with message management
- **Phase 3**: Advanced features (forking, templates, data entities)
- **Phase 4+**: Server deployment and multi-user support
